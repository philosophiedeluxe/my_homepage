Verstecktes Interface

Signal Index / Easter Egg Manual

Diese Seite sammelt die absichtlich versteckten Interaktionen der Homepage. Nichts davon ist notwendig, aber alles davon gehört zur Identität des Interfaces.

Nicht im Hauptmenü · über Footer-Signal oder Dev Console erreichbar

Discovery-Protokoll

Alle bekannten Signale

SESSION

Portfolio Boot

Trigger
Erster Seitenaufruf pro Session
Effekt
Kurzes Boot-Overlay mit Signal-Cut, danach Nav-Boot, Cursor-Handshake und Hero-Unlock.

HERO

Terminal Unlock

Trigger
Direkt nach dem Session-Boot
Effekt
Die Headline erscheint kurz als Code-Layer und rastet dann in echten Text ein.

CURSOR

Handshake

Trigger
Nach dem Boot-Reveal
Effekt
Der Custom Cursor meldet nacheinander INIT, AUTH und READY.

SHIFT + RIGHT CLICK

Cursor Context Menu

Trigger
Shift + Rechtsklick
Effekt
Öffnet ein eigenes Command-Menü mit Dev Console, Copy Link, Vita drucken, Mail und GitHub.

SECRET

Dev Console

Trigger
Ctrl + Alt + D oder Cursor-Menü
Effekt
Öffnet eine interne Mini-Konsole mit Route, Build, Stack-Hinweisen und verstecktem Signal-Link.

TEXT SURFACE

Keyword Cursor

Trigger
Hover auf technische Begriffe wie APEX, SQL, JavaScript, REST oder Software
Effekt
Der Cursor zeigt kontextuelle Kurzsignale wie APEX, SQL, FLOW oder {PK}.

IDLE

Cursor Sleep

Trigger
Maus bleibt etwa 12 Sekunden still
Effekt
Der Cursor wechselt in einen kleinen Sleep-Code-Zustand und wacht bei Bewegung wieder auf.

DWELL

Hero Terminal

Trigger
Hero bleibt etwa 7 Sekunden sichtbar
Effekt
Eine kurze Terminal-Zeile erscheint im Hero und sendet ein Cursor-Signal.

SECTION

Number Signals

Trigger
Klick auf dekorative Abschnittsnummern wie 01 bis 04
Effekt
Nummer, Cards und Content rebooten kurz mit Scan- und Daten-Jump-Effekt.

KEYBOARD

Matrix Rain

Trigger
matrix tippen
Effekt
Ein temporärer Matrix-Overlay läuft über die Seite und blendet danach selbst aus.

KEYBOARD

Manual Boot

Trigger
boot tippen
Effekt
Startet die lange interne Boot-Sequenz mit Terminalausgabe und Interface-Reveal.

KONAMI

Developer Mode

Trigger
↑ ↑ ↓ ↓ ← → ← → B A
Effekt
Aktiviert die große Developer-Konsole, Grid-Overlay, Modul-Outlines und typed Shell Output.

LANG

Language DEV Mode

Trigger
Sprachschalter sechsmal schnell anklicken
Effekt
Temporärer DEV-Sprachzustand mit monospace Interface-Treatment.

BRAND

Theme Shift

Trigger
Shift halten und auf das PK-Branding klicken
Effekt
Ein kurzer alternativer Chroma-Zustand verschiebt die Farbwelt der Seite.

SOURCE

DOM Signal

Trigger
DevTools oder HTML-Quelltext öffnen
Effekt
Ein versteckter Kommentar markiert die aktivierte Signal-Schicht.

PROFILE GRAPH

System Trace

Trigger
trace profile im Hero oder Command Palette
Effekt
Scant PROFILE_NODE, DELIVERY_MODULES, CAPABILITY_GRAPH, EXPERIENCE_LOG und TRUST_CHAIN als animierten Systemfluss.

COMMAND

Command Palette

Trigger
Ctrl + K
Effekt
Öffnet ein schnelles Portfolio-Command-Interface für Trace, Vita, Stack, Recruiter Mode, Mail, GitHub und Signal Index.

CAPABILITY

Live Skill Graph

Trigger
Skill-Knoten im Stack-Bereich anklicken
Effekt
Verbindet APEX, PL/SQL, JavaScript, Datenmodelle, Prozesse und Delivery mit kurzen Capability-Signalen.

DECISION MODE

Recruiter Mode

Trigger
Command Palette: Recruiter Mode
Effekt
Blendet eine kompakte Entscheidungsansicht ein: Passung, Profil-Unterschied und Belege.

EXPERIENCE LOG

Bootable Vita

Trigger
play timeline auf der Vita-Seite
Effekt
Spielt die Timeline als Log ab und fokussiert die beruflichen Stationen nacheinander.

Hinweis

Alle Effekte sind lokal, temporär und respektieren reduzierte Bewegung. Es werden keine externen Dienste, kein Tracking und keine Analytics ausgelöst.
